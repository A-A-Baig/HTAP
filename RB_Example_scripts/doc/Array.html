    matching_compound_members(procs_array) click to toggle source

    Takes an Array of Procs as an argument, and returns all members matching the criteria defined by each Proc via Array.matching_members. Note that it uses the ampersand to convert from Proc to block.

    # File matching_compound_members.rb, line 24
def matching_compound_members(procs_array)
  procs_array.map do |some_proc|
    # collect each proc operation
    matching_members(&some_proc)
  end.inject(self) do |memo,matches| 
    # find all the intersections, starting with self
    # and whittling down until we only have members 
    # that have matched every proc
    memo & matches
  end
end
