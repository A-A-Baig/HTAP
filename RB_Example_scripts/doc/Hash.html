    Given that self is a Hash with keys of XML tags and values of their respective counts in an XML source file, sort by the tag count, descending. Fall back to an ascending srt of the tag itself, weighted half as strongly.

    # File xml_tag_counter.rb, line 19
def sort_by_tag_count()
  self.sort do |a, b| 
    ( (b[1] <=> a[1]) * 2 ) + (a[0] <=> b[0])
  end
end

def sorted_by_tag_count()
  # sort_by_tag_count returns an Array of Arrays...
  sort_by_tag_count.inject({}) do |memo,pair|
    tag, count = pair
    memo.merge( { tag => count } )
  end
  # so we can re-Hash it with inject
end
